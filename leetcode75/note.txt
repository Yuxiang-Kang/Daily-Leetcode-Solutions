1. list --> string
result = " ".join(list)

2. To copy a list, do not use a = b. Instead, use "a = b[:]".
   Otherwise, b will change with a.

3. To create an arithmetic progression, use "range()".
   range(5,-1,-1) = [5,4,3,2,1,0]

4. To reverse a list, use "reverse()"
    a.reverse()

5. To break a string sentence into word strings, use "split()".
    string.split()

6. To get an infinity value, use "float('inf')".
